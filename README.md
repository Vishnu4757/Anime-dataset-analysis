# 📊 Anime Dataset Analysis

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on a dataset containing the **Top 15,000 Anime titles**.
The goal is to analyze trends in anime popularity, ratings, genres, and other key attributes to better understand viewer preferences and patterns in the anime industry.

Anime has become a global entertainment phenomenon, and analyzing such datasets helps uncover valuable insights about audience interests and content performance.

---

## 📂 Dataset Information

The dataset contains **15,000 anime records** with multiple attributes describing each title.

### Key Features in the Dataset

* **Anime ID** – Unique identifier for each anime
* **Name** – Original anime title
* **English Name** – English-translated title
* **Genre** – Categories of the anime
* **Type** – TV, Movie, OVA, etc.
* **Rank** – Ranking based on popularity or rating
* **Score** – User rating
* **Episodes** – Number of episodes
* **Popularity** – Viewer popularity ranking
* **Members** – Number of users who added the anime
* **Favorites** – Number of users who favorited the anime

---

## 🎯 Objectives

The main objectives of this project are:

* Understand the **structure and distribution of the dataset**
* Identify **top-ranked anime**
* Analyze **genre popularity**
* Explore **rating distributions**
* Detect **patterns and trends in anime performance**
* Perform **data cleaning and preprocessing**

## 📊 Exploratory Data Analysis (EDA)

The following steps were performed during analysis:

1. **Data Loading**

   * Imported the dataset using Pandas.

2. **Dataset Inspection**

   * Checked dataset shape
   * Viewed dataset structure and columns

3. **Data Cleaning**

   * Handling missing values
   * Formatting columns
   * Removing inconsistencies

4. **Statistical Analysis**

   * Summary statistics
   * Distribution of ratings and popularity

5. **Visualization**

   * Genre distribution
   * Rating trends
   * Popular anime categories
